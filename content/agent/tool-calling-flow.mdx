sequenceDiagram
  participant User
  participant Agent
  participant Tool
  participant LLM

  User->>Agent: 提交任务
  Agent->>LLM: 请求下一步计划
  LLM-->>Agent: 选择调用工具
  Agent->>Tool: 执行参数化调用
  Tool-->>Agent: 返回结构化结果
  Agent->>LLM: 汇总结果并请求最终回答
  LLM-->>User: 输出答案
</Mermaid>

<Explain>
### 核心价值
- 工具调用把“会说”扩展为“会做”。
- 结构化返回可降低 hallucination 风险。
- 通过循环步骤保留可观测性与可调试性。
</Explain>
